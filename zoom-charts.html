About project

This project goal was to create fully working dashboard demo using Zoom Charts as visualization tool inside Jedox. Jedox is an Enterprise Performance Management software solution for planning, analysis, reporting and optimization of value creation processes. Idea was to show power of Zoom Charts for data visualization inside Jedox.

Result

As the result I developed fully working dashboard using demo database available in Jedox. All dashboard elements are dependable on each other and interactable. This demo dashboard was used for promo video which was used in promo campaign for both ZoomCharts and Qbit. You can check it yourself in video bellow.

My Impact and Technologies Used

My impact in this project was integration of ZoomCharts and making dashboard look like on the mockup. Mockup was designed and provided by ZoomCharts.For this project were used two programing languages, JavaScript and PHP. JavaScript was used for using ZoomCharts SDK for adding charts to the Jedox. PHP is main Jedox API language and was used to get data from Jedox databases to display them in ZoomCharts later.
